# ScRotator

ScRotator is a simple screen rotation tool. When you run, it's turn your screen, that's it. It's only work for Windows for now

# PowerToys

I recommend using PowerToys Keyboard Manager Utility.
<img width="459" height="71" alt="resim" src="https://github.com/user-attachments/assets/fcd4aa6a-103b-4d59-9051-530626414545" />

1. download Python script (and install python, if you haven't)
2. Then install rotate-screen with `pip install rotate-screen` command
3. Then remap a shortcut via Keyboard Manager Utility <img width="770" height="131" alt="resim" src="https://github.com/user-attachments/assets/ebf1f9eb-06e4-4598-8760-cdc7eaf8af84" />
4. Give a shortcut input (like Win+PgDn)
5. Set action to `Run Program`
6. Set App to Python.exe path
7. Set Args as downloaded python script path
8. (Optional) If you don't wanna see opening terminal you can set visibility to `Hidden`
